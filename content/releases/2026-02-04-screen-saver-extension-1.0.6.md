## 🎉 [Screen Saver Extension] v1.0.6 릴리즈

> 📅 릴리즈 날짜: 2026-02-04

### 📝 기타 변경사항

- Chrome Web Store 신규 URL 차단 추가 (`chromewebstore.google.com`)
- Google 계정 관련 페이지 차단 추가
- `accounts.google.com` (로그인 페이지)
- `myaccount.google.com` (계정 관리 페이지)
- CLAUDE.md에 "제한된 페이지" 섹션 추가
- 제한되는 페이지 목록 (프로토콜별, 도메인별)
- 제한 사유 및 사용자 피드백 설명
- 구현 코드 예시 및 테스트 방법 포함
- \`chrome://\` - Chrome 내부 페이지
- \`chrome-extension://\` - 다른 확장 프로그램 페이지
- \`edge://\` - Edge 브라우저 내부 페이지
- \`about:\` - 브라우저 정보 페이지
- \`view-source:\` - 소스 보기 페이지
- \`chrome.google.com/webstore\` - Chrome Web Store (구 URL)
- \`chromewebstore.google.com\` - Chrome Web Store (신 URL) ✨ 새로 추가
- \`microsoftedge.microsoft.com/addons\` - Edge 확장 프로그램 스토어
- \`accounts.google.com\` - Google 계정 로그인 ✨ 새로 추가
- \`myaccount.google.com\` - Google 계정 관리 ✨ 새로 추가
