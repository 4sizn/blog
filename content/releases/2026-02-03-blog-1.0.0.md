## 🎉 [4sizn Blog] v1.0.0 릴리즈

> 📅 릴리즈 날짜: 2026-02-03

### ✨ 새로운 기능

- GitHub 릴리즈 자동 동기화 시스템 구현
- 템플릿 기반 릴리즈 노트 생성
- 스마트 릴리즈 노트 파싱

### 📝 기타 변경사항

- 매일 오전 9시 자동 실행 (GitHub Actions)
- 수동 실행 기능 지원
- 다중 저장소 추적 가능
- 코드와 템플릿 분리로 유지보수성 향상
- Placeholder 기반 동적 섹션 생성
- 비개발자도 템플릿 커스터마이징 가능
- Conventional Commits 스타일 자동 감지
- 섹션별 자동 분류 (Features, Bugfixes, Improvements 등)
- 빈 섹션 자동 제거
- 날짜 prefix 자동 추가 (YYYY-MM-DD 형식)
- 파일명과 타이틀에 일관된 날짜 표시
- 시간순 자동 정렬 지원
- 무한 리프레시 버그 수정
- permalink 이슈 해결
- 안정적인 페이지 로딩
- 포괄적인 개발 가이드 작성
- CLAUDE.md 파일에 상세 문서화
- 개발 환경 설정부터 배포까지 전체 가이드
- 문제 해결 섹션 포함
- **Frontend**: Quartz v4.5.2 (정적 사이트 생성)
- **Runtime**: Node.js 22+
- **CI/CD**: GitHub Actions
- **언어**: TypeScript, JavaScript (ESM)
- **자동화**: 릴리즈 동기화 스크립트
